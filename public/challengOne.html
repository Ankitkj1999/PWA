200 Days Coding Challenge Chapter One

Home Blog Challenge Chapter One

200 Days Coding Challeng C++ | DSA

Days 1-10 11-20 21-30

Its 15 September 2020 today, I am in my second year of engineering for the past one month. I have been taught C and C++ programming languages in my first and second semesters respectively. C was my first experience in programming, and because or obvious reasons, I was not able to have a firm hold on this programing language by the end of our course. And in the second semester due to Corvid-19, we had to skip half of our second semester. Again, I got only the basics covered in C++ too.

After my second semester, we have been given a break of around two and a half months. I invested all this precious time in learning how to make static websites and making static websites for others and postponing my main coding practice. I did earn Rs 8000/- form three clients, but now i doubt maybe I was wrong, I should have practice C - C++. From around the fourth semester, I may be giving interviews for internships or so.

Now moving forward to the third semester and I am being taught Data Structures & Algorithms, Computer Organization & Desing and Python. And due to a weak base in C and C++, I am lagging.
But whatever I have learned from it and now I am moving on. Still, I am in the third semester only, and I feel I can still make good progress.

Challenge Objectives

Now the idea is that I am going to do a coding challenge for 200 days (for now). In which I will practice around five to ten problems daily and mark the day in the table above. The problems are going to range from basics in C++ to DSA. Alongside I will be doing a 100 days python challenge as well.

1. Hello World

// Your First C++ Program
 
#include <iostream>
 
int main() {
    std::cout << "Hello World!";
    return 0;
}

Output

Hello World!

2. Program to Print Number Entered by Use

#include <iostream>
using namespace std;
 
int main()
{    
    int number;
 
    cout << "Enter an integer: ";
    cin >> number;
 
    cout << "You entered " << number;    
    return 0;
}

Output

Enter an integer: 23
You entered 23

3. Program to Add Two Numbers

#include <iostream>
using namespace std;
 
int main()
{
    int firstNumber, secondNumber, sumOfTwoNumbers;
 
    cout << "Enter two integers: ";
    cin >> firstNumber >> secondNumber;
 
    // sum of two numbers in stored in variable sumOfTwoNumbers
    sumOfTwoNumbers = firstNumber + secondNumber;
 
    // Prints sum 
    cout << firstNumber << " + " <<  secondNumber << " = " << sumOfTwoNumbers;     
 
    return 0;
}
 

Output

Enter two integers: 4
5
4 + 5 = 9

4. Program to Find Quotient and Remainder

#include <iostream>
using namespace std;
 
int main()
{    
    int divisor, dividend, quotient, remainder;
 
    cout << "Enter dividend: ";
    cin >> dividend;
 
    cout << "Enter divisor: ";
    cin >> divisor;
 
    quotient = dividend / divisor;
    remainder = dividend % divisor;
 
    cout << "Quotient = " << quotient << endl;
    cout << "Remainder = " << remainder;
 
    return 0;
}

Output

Enter dividend: 13
Enter divisor: 4
Quotient = 3
Remainder = 1

5. Program to Find Size of int, float, double and char in Your System

#include <iostream>
using namespace std;
 
int main() 
{    
    cout << "Size of char: " << sizeof(char) << " byte" << endl;
    cout << "Size of int: " << sizeof(int) << " bytes" << endl;
    cout << "Size of float: " << sizeof(float) << " bytes" << endl;
    cout << "Size of double: " << sizeof(double) << " bytes" << endl;
 
    return 0;
}

Output

Size of char: 1 byte
Size of int: 4 bytes
Size of float: 4 bytes
Size of double: 8 bytes
